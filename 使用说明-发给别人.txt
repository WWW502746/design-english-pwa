设计学专业英语学习工具 - 分发和桌面使用说明

一、发给别人电脑直接用
1. 把整个文件夹或压缩包发给对方。
2. 对方解压后双击 index.html 即可打开。
3. 如果想放到 Windows 桌面，右键运行“创建桌面快捷方式.ps1”，或在 PowerShell 中运行：
   powershell -ExecutionPolicy Bypass -File .\创建桌面快捷方式.ps1

二、发给别人手机并添加到桌面
1. 把这个文件夹上传到 HTTPS 静态网站空间，例如 GitHub Pages、Netlify、Cloudflare Pages、自己的服务器。
2. 用手机浏览器打开网站地址。
3. iPhone：点浏览器分享按钮，选择“添加到主屏幕”。
4. Android Chrome/Edge：点右上角菜单，选择“添加到主屏幕”或“安装应用”。
5. 第一次打开后会缓存应用文件，之后可以离线使用。

三、为什么不能只发一个本地文件就完整安装
手机浏览器通常要求 PWA 从 HTTPS 地址打开，才允许安装到桌面并启用离线缓存。
本地文件可以打开学习，但“像 App 一样添加到桌面”和离线缓存能力会受系统限制。

四、包含文件
- index.html：主程序
- manifest.json：手机桌面安装信息
- service-worker.js：离线缓存
- icons：桌面图标
- 创建桌面快捷方式.ps1：Windows 桌面快捷方式脚本
